Jak použít ikonu ve Windows

  1. Uložte .ico soubor do složky, která se nebude přesouvat (např. C:\Ikony).
  2. Na cílové složce klikněte pravým tlačítkem → Vlastnosti → karta Přizpůsobit.
  3. Klikněte na Změnit ikonu… → Procházet → vyberte váš .ico soubor → OK.
  4. Potvrďte Použít a OK. Pokud se ikona neaktualizuje, obnovte Průzkumníka (odhlásit/přihlásit nebo restartovat Explorer).
Tip: Sdílení přes URL uloží nastavení do hash části odkazu. Kdokoli s odkazem otevře stejný návrh.
